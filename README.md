# Selenium E-Commerce Automation Framework

## About
End-to-end test automation framework for an e-commerce web application
built using Selenium WebDriver and Java following Page Object Model design pattern.

## Tech Stack
- Java 17
- Selenium WebDriver 4.18
- Eclipse IDE
- Git & GitHub

## Test Scenarios Covered
- Valid login with correct credentials
- Products page load verification
- Add product to cart
- Cart page validation
- Complete checkout flow
- Order success message verification

## Project Structure
src/
  com.saucedemo.pages/
    LoginPage.java
    ProductPage.java
    CartPage.java
    CheckoutPage.java
  com.saucedemo.utils/
    BaseClass.java
  com.saucedemo.tests/
    MainTest.java

## How to Run
1. Clone the repository
2. Open in Eclipse as Java Project
3. Add Selenium JARs to Build Path
4. Right-click MainTest.java
5. Run As Java Application
